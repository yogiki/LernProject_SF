# Проект: Анализ вакансий из HeadHunter

## Основные метрики

- Всего вакансий: 49 197
- Работодателей: 23 501  
- Регионов: 1 362
- Вакансий в Data Science: 1 771 (3.6% от общего числа)
- Вакансий для Junior DS: 51

## Этапы анализа

1. **Предварительный анализ** — общие метрики базы данных
2. **Детальный анализ вакансий** — зарплаты, опыт, форматы работы
3. **Анализ работодателей** — активность компаний, географическое распределение
4. **Предметный анализ Data Science** — требования к навыкам, зарплатные ожидания

## Результаты

Детальные выводы по каждому этапу анализа представлены в соответствующих разделах [ноутбука](https://github.com/yogiki/LernProject_SF/blob/master/SF_Exam_3/Project_2_HH.ipynb), включая:
- Распределение вакансий по регионам
- Анализ зарплатных вилок
- Популярность форматов занятости
- Требования к опыту работы
- Активность работодателей в разных городах
- Ключевые навыки для Data Science специалистов
